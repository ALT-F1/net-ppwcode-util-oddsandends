Welcome to PPWCode.Util.OddsAndEnds!

PPWCode.Util.OddsAndEnds is a .NET library that is part of the PPWCode project. It consists of a collection of several small helpers and utility code.

Getting started

The current version of the library is mark II, designed to work with Microsoft .NET 4.5. The library is available as the NuGet package PPWCode.Util.OddsAndEnds.II in the NuGet gallery.

The API documentation of the library is available online here.

An older version of the library, mark I, is still available in maintenance mode. It was originally designed to work with Microsoft .NET 3.5 and later migrated to .NET 4.5. The mark I version of the library is available as the NuGet package PPWCode.Util.OddsAndEnds.I in the NuGet gallery.
